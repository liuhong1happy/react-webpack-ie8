# react-webpack-ie8

Make your `React` app work with `webpack` in `Internet Explorer 8.0`.

## Develop

1. clone the repository

        git clone https://github.com/liuhong1happy/react-webpack-ie8.git
    
2. install dependency package;

        npm install 

3. run development server

        npm start
        
4. build

        npm run build
